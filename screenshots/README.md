# Screenshots

Local screenshots and visual QA captures live here so the project root stays clean.

## Folders

- `browser-qa/YYYY-MM-DD/` - browser screenshots from local checks.
- `reference/site/` - full-site or homepage reference captures.
- `reference/blog/` - blog layout and article style references.
- `reference/assets/` - visual assets or banners used as design references.

## Naming

Use descriptive lowercase names:

```text
apps-desktop-after-card-update.png
home-mobile-nav-check.png
blog-wide-serif-desktop.png
```

## Git

Screenshot binaries are ignored by default. Keep them local unless a screenshot is intentionally part of a design/reference review, in which case force-add it explicitly.
